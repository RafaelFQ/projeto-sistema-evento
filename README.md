# Sistema Evento – Spring Boot + JPA

Projeto desenvolvido para o **Desafio: Modelo de Domínio e ORM** da **Formação Java Spring Profissional (DevSuperior)**.

O sistema gerencia **eventos acadêmicos**, permitindo o cadastro de **atividades**, seus **blocos de horários** e **participantes**, utilizando **Spring Boot**, **JPA/Hibernate** e **H2**.

---

## 🛠 Tecnologias

* Java 21
* Spring Boot 3.5.10
* Spring Data JPA / Hibernate
* Banco de dados H2
* Maven

---

## 🧩 Banco de Dados

* Criação automática das tabelas via JPA
* Dados iniciais inseridos por seeding
